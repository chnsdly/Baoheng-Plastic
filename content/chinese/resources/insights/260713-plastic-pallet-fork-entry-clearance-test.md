塑料托盘样品在纸面上看起来合适，到了仓库现场仍可能失败。外形尺寸可能符合路线要求，供应商报价中的载荷数据也可能看起来足够，但叉孔、底部筋位、底部支撑间距、托盘车轮路径、坡道过渡，或带载后的下挠，都可能与现场真实搬运设备不匹配。

批量确认前，先使用下面的记录表。它不能替代供应商图纸、正式载荷测试或现场安全审核。它的作用是把采购、仓库、维修和EHS团队需要看到的路线证据记录下来，再决定样品是否可以进入下一步。

## 先看搬运路线，而不是只看外形尺寸

不要因为样品只是“1200 x 1000 mm”或“可叉车使用”就批准采购。[塑料托盘](/zh/products/pallets/)必须适配实际叉齿、托盘车、地面、登车桥、坡道、操作人员和载荷工况。

英国HSE的[仓储安全指南](https://www.hse.gov.uk/pubns/priced/hsg76.pdf)支持用风险视角看待仓库搬运，美国OSHA的[工业车辆法规](https://www.osha.gov/laws-regs/regulations/standardnumber/1910/1910.178)适用于美国。这些资料不批准任何具体托盘型号，但它们支持一个实际判断：搬运设备、载荷和操作条件会影响适用性。

记录批准范围时要具体：已测试设备、已测试载荷、已测试路线、日期、操作人员和异常。如果后续要在另一种托盘车、坡道、货架、输送线、冷库或工厂使用，应单独验证。

## 现场记录1：移动样品前先记录设备

样品被叉起前先完成这一部分。使用现场实测值，不要只用目录假设。

| 记录项目 | 为什么重要 | 结果 |
|---|---|---|
| 叉车叉齿宽度、厚度、长度、间距和叉尖状态 | 叉齿可能能进入，但仍会碰到底部筋位，或在带载时伸入不足 | 通过 / 暂停 / 复测 |
| 手动或电动托盘车叉长、轮子布局和起升高度 | 即使叉车可进叉，托盘车轮子仍可能碰到支腿、底梁或底部筋位 | 通过 / 暂停 / 复测 |
| 必须使用的进叉方向 | 两向可进叉的托盘，不一定适合需要四向进叉的路线 | 通过 / 暂停 / 复测 |
| 地面、坡道、登车桥和门槛状态 | 很多小间隙问题出现在过渡位置，而不是平整测试地面 | 通过 / 暂停 / 复测 |
| 空载移动时的操作员反馈 | 强行操作、异常转向或反复撞击不应成为日常方法 | 通过 / 暂停 / 复测 |

如果路线经常使用手动或电动托盘车，请同时查看[托盘车适配检查](/zh/resources/insights/2629-plastic-pallet-pallet-jack-compatibility/)。叉车进叉和托盘车轮路径有关联，但不是同一个批准条件。

## 现场记录2：标出托盘接触点

装载前先检查样品底部。对于[货架型塑料托盘](/zh/products/pallets/rackable-plastic-pallets/)，重点看底梁连续性、叉孔和带载后的底梁接触。对于[九脚嵌套塑料托盘](/zh/products/pallets/nestable-plastic-pallets/)，重点看支腿间距、轮子路径，以及空托嵌套结构是否造成搬运接触。对于川字周边底或双面结构，要确认底部接触和实际需要的进叉方向。

请记录：

- 每个必须进叉方向的叉孔高度和宽度；
- 可见的筋位、底梁或支腿接触点；
- 空载搬运后的底部划痕；
- 每一种托盘车的轮子路径；
- 放下托盘时是否拖底；
- 影响进叉的飞边、损伤或变形。

标准可以帮助买方使用一致语言，但不能替代现场测试。[ISO 8611-1:2025](https://www.iso.org/standard/87543.html)涉及平托盘测试方法，也说明特定托盘设计的现场测试有价值。这里的ISO资料只能作为规格讨论背景，不能证明某一个托盘适合所有设备路线。

## 现场记录3：空载搬运检查

先让空托盘走完整个正常动作，再加货物。测试要受控；如果需要不安全用力或异常操作，应停止。

记录托盘是否：

- 从每个必须方向顺畅进入；
- 起升时没有叉尖或轮子接触异常；
- 行走时没有拖底、震动或轮子卡滞；
- 能通过日常最小转弯区域；
- 下降时不被底部筋位或底梁卡住；
- 停止和放下时保持稳定。

空载通过只表示可以继续测试，不等于批量采购批准。

## 现场记录4：带载路线检查

带载测试应代表正常路线，而不是最容易通过的条件。记录货物占位、估算重量、载荷分布、缠绕或固定方式、行走距离、起升高度和每个过渡点。

载荷类型必须分开。静载、动态搬运载荷和货架载荷是不同批准问题。这份间隙记录检查的是搬运适配性。如果路线涉及重载地堆、货架存放、长时间承载、冷库或敏感产品，应同时安排[批量采购前的塑料托盘载荷测试](/zh/resources/insights/2616-plastic-pallet-load-test-before-bulk-order/)。

出现以下情况时，应暂停样品批准：

- 带载后出现空载时没有的底部接触；
- 可见下挠减少叉齿或轮子间隙；
- 坡道、登车桥或门槛处反复撞击；
- 托盘倾斜或支撑不稳导致货物位移；
- 路线结束后出现产品、包装或托盘损伤；
- 操作员需要不安全或难以重复的变通动作。

如果同一托盘还要经过输送线，请先补充[输送线底部接触检查](/zh/resources/insights/2640-plastic-pallet-conveyor-bottom-contact-checks/)，不要假设托盘底部同时适配车辆和输送支撑。

## 批准结论：通过、暂停、改设计或澄清

批准语句要窄。“在A线、B型号车辆、C载荷、D路线和E/F进叉方向下通过”是有用记录。“适合所有仓库使用”不是。

| 状态 | 适用情况 | 下一步 |
|---|---|---|
| 按已测试路线通过 | 空载和带载检查可接受，没有异常或异常已受控 | 把条件写入采购规格 |
| 暂停，要求供应商澄清 | 间隙接近极限、接触原因不清或图纸缺失 | 要求当前图纸和型号假设 |
| 因路线变化复测 | 将使用不同车辆、坡道、载荷、温度、货架或输送线 | 单独运行现场记录 |
| 拒绝或重新设计 | 反复接触、强行进叉、不安全搬运或带载不稳定 | 下单前调整托盘结构、设备或路线 |

对于到货批次，请把批准范围接入[来料检验计划](/zh/resources/insights/2617-plastic-pallet-incoming-inspection-plan/)，让后续批次按同样关键尺寸和接触点检查。

## 批量下单前问供应商的问题

要求供应商提供当前图纸、叉孔尺寸、适配搬运设备假设、载荷定义、必要时的材料和温度限制、样品差异范围，以及类似应用中的路线观察。这些答案应支持现场记录，而不是替代现场记录。

如果你提供设备参数、载荷条件、进叉方向，以及样品路线照片或记录，宝衡可以协助评估托盘结构选项。负责任的批准仍应基于已测试现场条件、当前产品数据，以及你的安全和质量要求。

## 实用原则

把叉孔间隙视为现场观察，而不是目录承诺。只有当买方能说明测试了哪些设备、载荷、路线和搬运条件，并清楚批准边界在哪里，塑料托盘样品才适合进入批量批准。
